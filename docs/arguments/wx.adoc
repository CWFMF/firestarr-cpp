[[arg-wx]]
=== --wx

Specify input weather file. This requires the location to and the name of the file with it's extension.

Whole address:

`E:\Data\Burning Information\Weather.csv`

Relative address from current working directory:

`.\Data\Weather.csv`

[cols="1,3"]
|===
|Type |File path
|Required |Yes
|Manadtory Fields| `Date, Time, Temperature, Relative Humidity, Wind Speed, Wind Direction, Precipitation`
|===
